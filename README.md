# Indian Sign Language Recognition using CNN-BiLSTM

An AI-powered deep learning system for recognizing Indian Sign Language gestures using Convolutional Neural Networks (CNN) and Bidirectional Long Short-Term Memory (BiLSTM) networks to improve communication accessibility.

## Project Overview

Communication between hearing individuals and people who use Indian Sign Language (ISL) remains a significant challenge in everyday life. This project presents a deep learning-based sign language recognition system that combines Convolutional Neural Networks (CNN) for spatial feature extraction with Bidirectional Long Short-Term Memory (BiLSTM) networks for temporal sequence modeling. The objective is to accurately recognize sign language gestures and convert them into meaningful text, improving accessibility and communication.


## Objectives

- Develop an AI-based Indian Sign Language recognition system.
- Improve communication accessibility for deaf and hard-of-hearing individuals.
- Extract spatial features using CNN.
- Learn temporal gesture sequences using BiLSTM.
- Convert recognized gestures into readable text.


## Key Features

- Deep learning-based gesture recognition
- CNN feature extraction
- BiLSTM sequence modeling
- Real-time gesture interpretation
- Accessible communication support
